# DIET Gumla Website Clone

This is a frontend clone of the DIET Gumla website (https://jissoftware.in/dietgumla/) using HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Main HTML file
- `css/style.css` - Main styles
- `css/responsive.css` - Responsive styles
- `js/main.js` - JavaScript for interactivity
- `images/` - Image assets
- `favicon.ico` - Favicon

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.8
- Font Awesome 7.0.1
- Swiper 12

## How to Run

1. Open `index.html` in a web browser, or
2. Run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then open http://localhost:8000 in your browser.

## Features

- Responsive design
- Image carousel
- Mobile navigation menu
- Announcements marquee
- Faculty cards
- Gallery section
